# Changelog

## BETA-1.3

- Version updated to `BETA-1.3`
- Thin-jar distribution to reduce plugin artifact size
- PacketEvents moved to external dependency plugin
- Runtime libraries loaded via plugin metadata (`libraries`)
- Startup order adjusted for dependency stability (`POSTWORLD`)
- Bukkit-only fallback restriction mode when PacketEvents API is unavailable
- Expanded pre-auth action blocking coverage
- Premium permission gate removed for `/premium` and `/unpremium`
- Admin permission checks retained for `/authadmin` and `/lookup`
- English command metadata in plugin.yml
- English inline comments added to config.yml
